# How to build and execute

## C

Version - C99

```
$ gcc -o yes yes.c
$ ./yes
```

OR

```
$ ./yes argument list
```

## C++

```
$ g++ -o yes yes.cpp
$ ./yes
```

OR

```
$ ./yes argument list
```

## Java

Version - Java 8

```
$ javac yes.java
$ java yes
```

OR

```
$ java yes argument list
```

## Node.js

```
$ node yes.js
```

OR

```
$ node yes.js argument list
```
