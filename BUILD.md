# How to build and execute

## C

Version - C99

```shell
$ gcc -o yes yes.c
$ ./yes
```

OR

```shell
$ ./yes argument list
```

## C++

```shell
$ g++ -o yes --std=c++11 yes.cpp
$ ./yes
```

OR

```shell
$ ./yes argument list
```

## Java

Version - Java 8

```shell
$ javac yes.java
$ java yes
```

OR

```
$ java yes argument list
```

## Groovy

```shell
$ ./yes.groovy
```

OR

```
$ ./yes.groovy argument list
```

## Node.js

```shell
$ node yes.js
```

OR

```
$ node yes.js argument list
```

## Objective-C

```shell
$ clang -framework Foundation -o yes yes.m
$./yes
```

OR

```shell
$ ./yes argument list
```

## Vala

```shell
$ valac -o yes yes.vala
$./yes
```

OR

```shell
$ ./yes argument list
```

## R

```shell
$ yes() 
```

OR

```shell
$ yes(string argument)
```

## Pure Data

```shell
$ pd -batch -open yes.pd
```

OR

```shell
$ pd -batch -open yes.pd -send "_ argument list"
```
